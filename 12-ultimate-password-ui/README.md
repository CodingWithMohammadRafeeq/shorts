## нҫҜ How It Works

* When typing password:

  * Weak вҶ’ нҙҙ Red bar
  * Medium вҶ’ нҝ  Orange bar
  * Strong вҶ’ нҝў Green full bar

* Clicking the eye icon:

  * нұҒ Opens password (left вҶ’ right shine)
  * н№Ҳ Hides password (right вҶ’ left shine)
  * вңЁ Includes smooth blink animation
