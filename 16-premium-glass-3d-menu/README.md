## нҫҜ How It Works

* Each menu item has two sides:

  * Front face
  * Back face
* On hover:

  * The inner container rotates using `rotateY(180deg)`
* 3D effect is created using:

  * `perspective`
  * `transform-style: preserve-3d`
  * `backface-visibility`
